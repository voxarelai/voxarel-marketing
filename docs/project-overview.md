# Project Overview

## Voxarel Marketing Website

### Purpose
A modern, high-converting marketing website showcasing Voxarel's AI-powered logistics platform for UAE freight forwarding operations.

### Project Details

| Attribute | Value |
|-----------|-------|
| **Project Name** | voxarel-marketing |
| **Type** | Marketing/Landing Page |
| **Framework** | Next.js 15.1.0 |
| **Language** | TypeScript |
| **Status** | Active Development |

### Target Audience
- Logistics company executives (UAE)
- Freight forwarding decision-makers
- Operations managers
- Business development teams

### Key Features Showcased

#### 1. Predictive Container Optimization
AI-powered forecasting of container fill rates 3-5 days in advance, enabling proactive booking and eliminating overflow delays.

#### 2. Multi-Package Smart Cart
Single-flow package collection for field agents with unified payment processing for multiple packages per shipment.

#### 3. Real-Time WhatsApp Tracking
Automated customer notifications at every milestone, reducing support calls by 80%.

#### 4. Executive Intelligence
Real-time dashboards for branch performance, container utilization, and revenue trends.

### Website Sections

| Section | Purpose |
|---------|---------|
| **Navigation** | Sticky header with product/company links and CTAs |
| **Hero** | Full-screen introduction with value proposition |

| **Footer CTA** | Final conversion opportunity |
| **Footer** | Links and legal |

### Design System

**Theme**: Dark mode with glassmorphism effects
**Accent Color**: Orange (#f97316)
**Typography**:
- Headlines: Playfair Display (serif)
- Body: Inter (sans-serif)

**Visual Style**:
- Gradient mesh backgrounds
- Animated gradient orbs
- Frosted glass cards
- Smooth scroll animations
