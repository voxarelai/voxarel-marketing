## Critical Directories

| Directory | Purpose | Key Files |
|-----------|---------|-----------|
| `src/app/` | Next.js App Router | `layout.tsx` (entry), `page.tsx` (home), `globals.css` |
| `src/components/` | Marketing sections | 12 section components + barrel export |
| `src/components/ui/` | Reusable primitives | Button, Card, Badge, Progress, Tabs |
| `src/lib/` | Utilities | `cn()` class name merger |
| `public/` | Static assets | Logos, hero image |
| `docs/` | Documentation | Generated by this workflow |

## Entry Points

| File | Purpose |
|------|---------|
| `src/app/layout.tsx` | Root layout - fonts (Inter, Playfair), metadata, dark theme |
| `src/app/page.tsx` | Home page - composes all marketing sections |
| `src/app/globals.css` | Global styles - theme variables, glassmorphism, animations |
